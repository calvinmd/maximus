# Lever


A DeFi Lending tool and sdk.


## Dependencies

- Node 8.12.0 + npm
- Ethereum local node / Ganache server


## File Structure

```
    .
    ├── contracts - Truffle contracts
    ├── migrations - Truffle migrations
    └── test - Truffle tests

```


## Install

```bash
npm install
```


## Migrate

Assumes running ganache server.

```bash
npm run migrate
```
